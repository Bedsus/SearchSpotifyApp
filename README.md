# SearchSpotifyApp
============================================
Реализация умного поиска музыкальной библиотеки Spotify

### Основные характеристики приложения:
* Минимальная версия SDK: 21 (5.0)
* Язык разработки: [Kotlin 1.4.10](https://kotlinlang.org/)
* Сборка проекта: [Gradle 5.4.1](https://gradle.org/)
* Среда разработки: [Android Studio 3.5.2](https://developer.android.com/studio)
* Архитектура: MVVM+R
* Разрешения: Интернет

### Используемые библиотеки:
* [Retrofit](https://github.com/square/retrofit) - для сетевых запросов
* [Koin](https://github.com/InsertKoinIO/koin) - внедрение зависимостей
* [Coroutines](https://github.com/Kotlin/kotlinx.coroutines) - асинхронное и неблокирующее программирование
